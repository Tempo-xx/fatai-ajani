Overview

A Shopify store was failing all three Core Web Vitals metrics — directly impacting both Google search ranking and on-site conversion rate. A complete technical performance audit was conducted and all issues resolved, bringing every metric into the passing range.

Fixes Applied

  • LCP resolved by optimising the largest above-fold image — correct format, preloading, and CDN delivery
  • CLS resolved by adding explicit dimension attributes to all images and embeds
  • INP resolved by deferring non-critical JavaScript and removing render-blocking third-party scripts
  • Font loading optimised using font-display:swap and preconnect directives
  • App scripts audited — three redundant Shopify apps removed, reducing total JavaScript payload by 40%

How I work

The Engagement Process

Every project follows the same four-stage framework. Diagnosis before action. Measurement throughout. Compounding improvement over time.

01

Audit & Diagnose

A complete review of traffic, funnel performance, UX, ads, email, and technical health — identifying exactly where revenue is being lost before any work begins.

02

Prioritise by Impact

All findings are ranked by impact-to-effort ratio. The highest-leverage fixes are addressed first so results are visible within weeks, not months.

03

Execute & Test

Every change is implemented with testing discipline. Nothing is assumed to work — winning approaches are scaled, underperforming ones are removed quickly.

04

Report & Compound

Structured reporting with clear business context — not raw numbers. Each cycle builds on the previous, compounding improvements across the full customer lifecycle.
